Web Resources

Below is a list of websites that may be useful for learning Responsive Web Design

  • W3Schools
    W3 Schools is great! This is where I learned almost everything I know about HMTL/CSS/JS. The tutorials are clear and to-the-point.

  • Bootstrap
    Bootstrap's homepage is full of documentation and examples. Of course, the website itself is built using the Bootstrap framework, so sometimes the best trick is just to view the page source to see how it's done.

  • Daring Fireball
    John Gruber has blogged and podcasted about design for decades. He has strong opinions about product design, web design, and software design.

  • Wikipedia
    Web designers love to build web pages, so Wikipedia pages about web design get lots of love. The page about Responsive Web Design alone has over 50 references.

  • Ars Technica
    Ars Technica is a lesser-known Conde Nast publication, but is possibly the best technology magazine on the internet. Searching for "Responsive Web Design" produces hundreds of useful articles.

  • Lynda.com
    Lynda publishes the best video training materials in the technology industry. There is plenty of content around front-end development tools like Bootstrap.

  • Google - Responsive Web Design Basics
    I found this article, unsurprisingly, by googling around. It is a very well-organized and thorough introduction to Responsive Web Design. Google describes their Web Fundamentals pages as their "opinionated reference for building amazing web experiences."
